# ISO 27001:2022 Annex A Controls — SecureTech

> 📖 **Instructions:** ISO 27001:2022 has 93 controls organized into 4 themes. You don't need to document all 93 — focus on the most relevant controls for a cybersecurity startup. Aim for 6-10 controls per theme (covering the most impactful ones).
>
> For each control, document what it requires, SecureTech's current state, and any notes.
